# PersistAI

Persistent memory for AI agents. Solve context limits and maintain knowledge across sessions.

## Tech Stack

- Node.js (TypeScript)
- PostgreSQL (pgvector)
- Redis
- Express

## Features

- Multi-tier memory (Working/Short-term/Long-term)
- Semantic search retrieval
- Automatic memory extraction
- Context window stress testing (100K+ tokens)
- MCP Server Integration: Plug directly into Claude Desktop or other agents.
- Dreaming & Consolidation: Background optimization of memory storage.
